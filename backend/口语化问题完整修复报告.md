## ✅ 实施的解决方案

### 1. 扩充同义词映射表

从最初的**8个**映射扩充到**143个**映射，涵盖教育领域的所有常见口语化表达。

#### 核心映射示例

```javascript
// 成绩相关
'挂科' → '不及格'
'挂了' → '不及格'
'考砸了' → '成绩不合格'

// 处理相关
'怎么办' → '如何处理'
'怎么申请' → '申请流程'
'去哪办' → '办理地点'

// 疑问词汇
'能不能' → '是否可以'
'行不行' → '是否可行'
'好不好' → '是否合适'
```

### 2. 优化问题标准化函数

```javascript
function normalizeQuestion(question) {
    let normalized = question;

    // 替换所有同义词
    for (const [colloquial, formal] of Object.entries(SYNONYM_MAP)) {
        const regex = new RegExp(colloquial, 'g');
        normalized = normalized.replace(regex, formal);
    }

    return normalized;
}
```

### 3. 降低检索阈值

- **修复前**：minScore = 0.5
- **修复后**：minScore = 0.3
- **效果**：增加相关文档的召回率

### 4. 集成到问答接口

在 `src/routes/rag.js` 中自动应用标准化：

```javascript
router.post('/answer', authenticate, async (req, res) => {
    const { question, options = {} } = req.body;

    // 自动标准化
    const normalizedQuestion = normalizeQuestion(question);

    // 使用标准化后的问题进行检索
    const result = await ragService.answer(normalizedQuestion, {
        ...options,
        minScore: options.minScore || 0.3
    });

    // 返回结果
    res.json({
        success: true,
        data: {
            ...result,
            originalQuestion: question,
            normalizedQuestion: normalizedQuestion !== question ? normalizedQuestion : undefined
        }
    });
});
```
