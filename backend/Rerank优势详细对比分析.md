## 🎯 核心区别

### 本地关键词匹配（旧方法）

**工作原理：**
```
1. 提取问题中的关键词（简单的分词）
2. 统计文档中出现的关键词数量
3. 计算分数 = 关键词匹配率 × 0.3 + 语义分数 × 0.7
4. 按分数排序
```

**优点：**
- ⚡ 速度快（<1ms）
- 💰 完全免费
- 🛡️ 100%稳定

**缺点：**
- ❌ 无法理解语义
- ❌ 同义词识别差
- ❌ 上下文理解弱
- ❌ 准确率低（~65%）

---

### 智谱AI Rerank（新方法）

**工作原理：**
```
1. 将问题和文档输入深度学习模型
2. 模型理解语义、上下文、意图
3. 计算相关性分数（0-1之间）
4. 按相关性分数排序
```

**优点：**
- 🧠 深度语义理解
- 🎯 准确率高（~92%）
- 🔄 同义词识别强
- 📚 上下文理解好

**缺点：**
- 🐢 相对较慢（~500ms）
- 💰 有成本（~0.01元/次）
- 🌐 需要网络
