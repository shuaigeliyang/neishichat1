## 📋 问题回顾

**用户问题**："学生挂科了会怎么样"
**系统回答**："抱歉，根据学生手册，没有找到与您问题相关的信息。"
**实际问题**：文档中有65处相关内容，但检索失败

---

## 🔍 根本原因

1. **语言风格差异**
   - 用户使用口语："挂科"
   - 文档使用正式用语："不及格"

2. **语义匹配不足**
   - 向量模型对"挂科"和"不及格"的语义相似度计算不够准确
   - minScore阈值（0.5）过高，过滤掉了相关文档

---

## ✅ 实施的修复方案

### 1. 添加同义词映射表

在 `src/routes/rag.js` 中添加：

```javascript
const SYNONYM_MAP = {
    // 成绩相关
    '挂科': '不及格',
    '挂了': '不及格',
    '没过': '不及格',
    '考砸了': '成绩不合格',

    // 补考相关
    '重考': '补考',

    // 重修相关
    '重读': '重修',

    // 处理相关
    '怎么办': '如何处理',
    '怎么样': '后果',

    // 其他
    '能不能': '是否可以'
};
```

### 2. 实现问题标准化函数

```javascript
function normalizeQuestion(question) {
    let normalized = question;

    // 替换所有同义词
    for (const [colloquial, formal] of Object.entries(SYNONYM_MAP)) {
        const regex = new RegExp(colloquial, 'g');
        normalized = normalized.replace(regex, formal);
    }

    return normalized;
}
```

### 3. 修改问答接口

```javascript
router.post('/answer', authenticate, async (req, res) => {
    // ... 前面的代码

    // 标准化问题
    const normalizedQuestion = normalizeQuestion(question);

    // 调用问答服务，使用更低的minScore
    const result = await ragService.answer(normalizedQuestion, {
        ...options,
        minScore: options.minScore || 0.3  // 从0.5降低到0.3
    });

    // ... 返回结果
});
```
