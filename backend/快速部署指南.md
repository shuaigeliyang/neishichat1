# 🎉 本地Embedding服务部署指南（简化版）

## ⚡ 快速开始（3步）

### 方法1：使用一键启动脚本（推荐）✅

**Windows用户：**

1. 双击运行 `一键启动本地Embedding.bat`
2. 等待依赖自动安装
3. 等待模型下载（首次启动，约500MB）
4. 看到服务启动信息

**完成！** 服务将在 http://localhost:5001 运行

---

### 方法2：手动安装（如果脚本失败）

**步骤1：安装Python**

1. 下载Python：https://www.python.org/downloads/
2. 安装时勾选 "Add Python to PATH"
3. 验证安装：打开命令提示符，运行 `python --version`

**步骤2：安装依赖**

打开命令提示符（cmd），进入backend目录：

```bash
cd E:\外包\教育系统智能体\backend
python -m pip install flask flask-cors sentence-transformers numpy
```

**步骤3：启动服务**

```bash
python local_embedding_service.py
```

---

## 🔍 如何验证服务是否成功

看到以下信息表示成功：

```
✓ 模型加载完成！
   向量维度: 384
 * Running on http://0.0.0.0:5001
```

**测试服务：**

打开浏览器访问：http://localhost:5001/health

应该看到：
```json
{
  "status": "ok",
  "service": "Local Embedding Service (Python)",
  "model": "paraphrase-multilingual-MiniLM-L12-v2"
}
```

---

## 🎯 完成部署后

### 1. 保持Python服务运行

不要关闭命令提示符窗口，或者使用后台运行：

```bash
start /B python local_embedding_service.py
```

### 2. 启动后台服务

新开一个命令提示符：

```bash
cd E:\外包\教育系统智能体\backend
npm start
```

### 3. 测试RAG服务

打开浏览器，访问前端，提问测试！

## ❓ 常见问题

**Q1: 双击bat文件后闪退**

A1: 右键点击bat文件，选择"编辑"，检查是否有Python

**Q2: 提示"python不是内部命令"**

A2: Python未安装或未添加到PATH，请重新安装Python

**Q3: SSL证书错误**

A3: 使用命令提示符（cmd）而不是PowerShell

**Q4: 模型下载慢**

A4: 正常现象，模型约500MB，请耐心等待
