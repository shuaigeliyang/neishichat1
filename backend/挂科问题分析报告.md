
### 2. 语言风格差异 ❌

这是问题的根源！

| 用户提问 | 文档用语 | 匹配度 |
|---------|---------|--------|
| 挂科 | 不及格、成绩不合格 | ⚠️ 低 |
| 怎么样 | 后果、处理办法、规定 | ⚠️ 低 |

**口语化** vs **正式用语**的语义鸿沟！

### 3. 可能的技术原因

1. **向量匹配问题**
   - "挂科"和"不及格"的语义向量可能不够接近
   - 即使使用智谱AI的embedding模型，口语和正式用语的匹配度也可能较低

2. **minScore阈值过高**
   - 当前设置：`minScore: 0.5`
   - 可能相关文档的分数在0.4-0.49之间，被过滤掉了

3. **检索模式问题**
   - 可能Rerank未生效
   - 或者本地关键词匹配没有提取到"挂科"这个关键词

## 💡 解决方案

### 方案1：降低minScore阈值（快速修复）

```javascript
const result = await ragService.answer(question, {
    minScore: 0.3,  // 从0.5降低到0.3
    topK: 5,
    useReranking: true
});
```

### 方案2：添加同义词映射（推荐）

在检索前将口语化词汇转换为正式用语：

```javascript
// 添加到检索引擎
normalizeQuery(query) {
    const synonyms = {
        '挂科': '不及格',
        '挂了': '不及格',
        '没过': '不及格',
        '考砸了': '成绩不合格',
        '重考': '补考',
        '怎么办': '如何处理',
        '怎么样': '后果',
        '怎么处理': '处理办法'
    };

    let normalizedQuery = query;
    for (const [colloquial, formal] of Object.entries(synonyms)) {
        normalizedQuery = normalizedQuery.replace(new RegExp(colloquial, 'g'), formal);
    }

    return normalizedQuery;
}
```

### 方案3：优化关键词提取（长期方案）

改进本地关键词匹配，识别同义词：

```javascript
extractKeywords(text) {
    const keywords = [];

    // 原有关键词提取
    const words = text.match(/[\u4e00-\u9fa5]{2,}/g) || [];

    // 添加同义词映射
    const synonymMap = {
        '挂科': ['不及格', '成绩不合格'],
        '补考': ['重考', '重新考试'],
        '重修': ['重新学习', '重读']
    };

    // 扩展关键词
    for (const word of words) {
        keywords.push(word);
        if (synonymMap[word]) {
            keywords.push(...synonymMap[word]);
        }
    }

    return keywords;
}
```

### 方案4：混合查询策略（最佳方案）

```javascript
async retrieve(question, options = {}) {
    const results = [];

    // 1. 原始查询
    results.push(await this.search(question, options));

    // 2. 标准化后的查询
    const normalized = this.normalizeQuery(question);
    if (normalized !== question) {
        results.push(await this.search(normalized, options));
    }

    // 3. 合并去重
    return this.mergeResults(results);
}
```
