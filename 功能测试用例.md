## 💬 上下文功能测试

### 测试目标
验证AI能够记住对话历史，支持追问和代词引用。

### 测试用例 1：连续追问

**第1轮对话：**
```
转专业需要什么条件？
```

**预期回复：** AI应该回答转专业的条件，可能包括：
- 成绩要求
- 时间限制
- 申请流程等

**第2轮对话（追问）：**
```
那具体怎么申请呢？
```

**预期回复：** AI应该理解"那具体怎么申请"是在问转专业的申请流程
- ✅ 回答转专业的申请流程
- ✅ 包含申请步骤、所需材料等

**第3轮对话（继续追问）：**
```
需要准备哪些材料？
```

**预期回复：** AI应该理解这是在问转专业需要的材料
- ✅ 列出所需材料清单
- ✅ 可能包含申请表、成绩单等

---

### 测试用例 2：代词引用

**第1轮对话：**
```
重修的流程是什么？
```

**预期回复：** AI回答重修的流程

**第2轮对话（代词引用）：**
```
它的费用是多少？
```

**预期回复：** AI应该理解"它"指的是"重修"
- ✅ 回答重修的费用
- ✅ 不是回答其他无关内容

**第3轮对话（代词引用）：**
```
在哪里办理？
```

**预期回复：** AI应该理解是在问重修在哪里办理
- ✅ 回答办理地点和部门

---

### 测试用例 3：多主题切换

**第1轮对话：**
```
奖学金有哪些类型？
```

**预期回复：** AI列举奖学金类型

**第2轮对话（切换话题）：**
```
那休学需要什么条件？
```

**预期回复：** AI应该理解这是一个新话题（休学），而不是奖学金
- ✅ 回答休学的条件
- ❌ 不是继续讲奖学金

**第3轮对话（回到之前话题）：**
```
国家奖学金的申请条件是什么？
```

**预期回复：** AI应该记得之前讨论过奖学金
- ✅ 回答国家奖学金的申请条件

---

### 测试用例 4：上下文过期

**第1轮对话：**
```
考试怎么报名？
```

**预期回复：** AI回答考试报名流程

**等待30分钟...**

**第2轮对话：**
```
具体什么时间？
```

**预期回复：**
- ⚠️ 可能不记得之前的内容（上下文已过期）
- ✅ AI会询问用户指的是什么考试

## 📝 测试注意事项

### 测试顺序建议

1. **先测试管理员** - 确保所有功能可用
2. **再测试教师** - 验证权限限制
3. **最后测试学生** - 验证最严格的权限
4. **单独测试上下文** - 专门测试对话记忆
5. **批量测试表单** - 逐个测试22个表单

### 常见问题处理

**Q1: 登录失败怎么办？**
- 检查用户名和密码是否正确
- 确认后端服务是否运行（http://localhost:3000/health）
- 查看浏览器控制台错误信息

**Q2: 上下文不生效？**
- 确保在同一个会话中对话
- 不要刷新页面（刷新会清除上下文）
- 检查是否超过30分钟（会自动过期）

**Q3: 表单下载失败？**
- 检查浏览器是否阻止了下载
- 查看后端日志
- 确认exports/forms目录存在

**Q4: 权限验证不准确？**
- 确认登录的角色正确
- 查看后端日志中的权限验证记录
- 检查system_logs表
