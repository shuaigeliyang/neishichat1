## 🎯 CSV导出插件使用指南

**插件名称**：CSV导出工具（自定义插件）

**插件功能**：将数据导出为CSV格式文本，用户复制保存

**依赖包**：无（使用Python内置库，速度极快）

### 插件参数说明（全部为String类型）

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| data | String | ✅ | JSON字符串格式的数据数组 |
| filename | String | ❌ | 文件名（不含扩展名），默认：export |
| headers | String | ❌ | JSON字符串格式的表头映射，无映射时传 "{}" |

### 插件返回结果

| 字段名 | 类型 | 说明 |
|--------|------|------|
| success | Boolean | 是否成功 |
| message | String | 结果消息 |
| filename | String | 生成的文件名（.csv） |
| record_count | Integer | 导出的记录数 |
| csv_content | String | CSV文本内容（用户复制保存） |

### 如何向用户展示结果

**⚠️ 重要：不要在对话中打印csv_content的内容！**

插件返回后，**直接告诉用户文件已生成**，不要把csv_content的内容显示出来：

```
✅ CSV文件已生成！

📄 文件名：学生信息.csv
📊 共导出 X 条记录

💾 **请复制下方代码块内容，保存为.csv文件即可用Excel打开**
```

**关键点**：
1. **不要逐条打印数据**
2. **不要在对话中显示csv_content的原始内容**
3. 只显示文件名和记录数
4. 把csv_content放在代码块中供用户复制

### 使用步骤

**步骤1：查询数据库获取完整数据**
```sql
SELECT * FROM 学生信息 WHERE 学院='计算机科学与技术学院'
```

**步骤2：将数据转换为JSON字符串格式**

查询结果：
```
学号: 1, 姓名: 陈丽, 性别: 女, 学院: 计算机科学与技术学院
学号: 2, 姓名: 周静, 性别: 女, 学院: 计算机科学与技术学院
```

转换为JSON字符串（注意：必须是字符串格式，用引号包裹）：
```
"[{\"学号\":1,\"姓名\":\"陈丽\",\"性别\":\"女\",\"学院\":\"计算机科学与技术学院\"},{\"学号\":2,\"姓名\":\"周静\",\"性别\":\"女\",\"学院\":\"计算机科学与技术学院\"}]"
```

**步骤3：调用CSV导出插件**

```
调用插件：CSV导出工具

参数设置：
- data: "[{\"学号\":1,\"姓名\":\"陈丽\",\"性别\":\"女\"}]"
- filename: "学生信息"
- headers: "{}"
```

**步骤4：向用户展示结果**

```
✅ CSV文件已生成！

📄 文件名：学生信息.csv
📊 共导出 X 条记录

� 请复制下方代码块内容保存
```

## ⚠️ 重要提醒

**参数格式要求**：
- ✅ data参数必须是**JSON字符串**，用双引号包裹
- ✅ JSON字符串内部的双引号需要**转义**（\"）
- ✅ headers参数传 `"{}"` 表示使用原始字段名
- ✅ filename不需要加扩展名，插件会自动添加.csv

**正确示例**：
```
data: "[{\"学号\":1,\"姓名\":\"张三\"}]"
```

**错误示例**：
```
data: [{"学号":1,"姓名":"张三"}]  ❌ 不是字符串
data: '[{"学号":1}]'  ❌ 单引号在某些环境可能有问题
```
