## 3. 执行流程

### 3.1 创建智能分块脚本

**文件**：`create_chunks_new.py`

**功能**：
- 基于高质量数据源创建优化的文档分块
- 智能分块策略（段落优先、句子重叠）
- 目标分块大小：1000字符
- 分块重叠：200字符
- 生成252个文档块

**统计**：
- 总分块数：252
- 平均大小：850字符
- 页码范围：1~128

### 3.2 生成向量索引

**文件**：`backend/generate_embeddings_new.js`

**功能**：
- 使用智谱AI embedding-3模型（2048维）
- 请求队列机制（避免限流）
- 缓存优化（二次生成31秒完成）
- 兼容旧格式的字段（indexed、timestamp）

**统计**：
- 向量维度：2048
- 文件大小：11.74MB
- 生成时间：140秒（首次）/ 31秒（缓存）
- 向量缓存：3795条记录

## 5. 技术要点

```json
{
  "metadata": {
    "total_chunks": 252,
    "total_pages": 4735,

  },
  "chunks": [
    {
      "chunk_id": 1,
      "page_num": 1,

      "char_count": 1129,

```

**retrieval_index.json**：
```json
{

```

3. **队列管理**：避免API限流、指数退避重试
4. **降级方案**：API失败时自动降级到本地方案
