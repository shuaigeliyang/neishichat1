### 2. 修复接口查询语句

**文件：** `backend/src/routes/teachers.js`

**修改前：**
```javascript
SELECT
  co.offering_id,
  c.course_name,
  c.course_code,
  c.credits,
  c.course_type,
  co.class_id,
  cl.class_name,
  co.semester,
  co.schedule,
  co.classroom,
  co.current_students,  // ❌ 字段不存在
  co.max_students        // ❌ 字段不存在
FROM course_offerings co
LEFT JOIN courses c ON co.course_id = c.course_id
LEFT JOIN classes cl ON co.class_id = cl.class_id
WHERE co.teacher_id = ? AND co.status = 1
```

**修改后：**
```javascript
SELECT
  co.offering_id,
  c.course_name,
  c.course_code,
  c.credits,
  c.course_type,
  co.class_id,
  cl.class_name,
  co.semester,
  co.schedule,
  co.classroom,
  co.status              // ✅ 保留状态字段
FROM course_offerings co
LEFT JOIN courses c ON co.course_id = c.course_id
LEFT JOIN classes cl ON co.class_id = cl.class_id
WHERE co.teacher_id = ?
```

**删除了不存在的字段：**
- ❌ current_students（字段不存在）
- ❌ max_students（字段不存在）

**删除了status条件：**
- 原来有 `AND co.status = 1`，可能导致过滤掉数据
