## 🎯 可能的原因分析

### 原因A：RAG检索结果不同

**问题1：** "挂科了会怎么样"
```
RAG检索：找到0个文档
返回：success: true, data: { answer: "抱歉，没有找到..." }
前端显示："抱歉，根据学生手册，没有找到与您问题相关的信息..."
```

**问题2：** "我的学生挂科了会怎么样"
```
RAG检索：找到N个文档（N > 0）
返回：success: true, data: { answer: "根据学生手册...", sources: [...] }
前端显示：RAG的回答（有来源信息）
```

**为什么会不同？**
- 问题标准化后的结果不同
- "挂科了会怎么样" → "学生不及格了会后果"
- "我的学生挂科了会怎么样" → 可能保留了"学生"，更匹配文档

---

### 原因B：一个降级到智谱AI了

**问题1：** "挂科了会怎么样"
```
RAG检索：找到0个文档
返回：success: true, data: { answer: "抱歉，没有找到..." }
前端显示：RAG的回答
```

**问题2：** "我的学生挂科了会怎么样"
```
RAG请求：抛出异常（超时/网络错误）
前端catch到错误
降级到：handleChatQuery（智谱AI）
前端显示：智谱AI的回答（没有来源信息）
```

**为什么会不同？**
- 第二个问题更长，处理时间更长，可能超时
- 或者遇到了其他异常

---
