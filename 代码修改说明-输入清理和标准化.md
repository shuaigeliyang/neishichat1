## 📋 问题描述

### 错误现象

用户输入："我的我的我的成绩"

**系统处理：**
```
1. 意图识别：包含"我的成绩" → 识别为数据库查询 ✅

```

**问题：**
- ❌ 重复的"我的"可能导致AI生成错误的SQL
- ❌ 查询失败后降级到普通聊天
- ❌ 智谱AI可能编造虚假信息
- ❌ 用户体验差

**预期行为：**
```
1. 输入清理："我的我的我的成绩" → "我的成绩"
2. 意图识别：数据库查询 ✅
3. 后端处理：生成正确的SQL ✅
4. 返回结果：显示真实成绩 ✅
```

## 🔧 修改内容

### 修改1：添加输入清理函数

**文件：** `frontend/src/pages/Chat.jsx`
**位置：** 在messagesEndRef定义之后

**新增函数：**

```javascript
/**
 * 清理和标准化用户输入
 * 处理重复的词语、多余的空格等
 */
const cleanInput = (text) => {
  if (!text) return text;

  let cleaned = text;

  // 1. 去除重复的"我的"（如："我的我的我的成绩" → "我的成绩"）
  cleaned = cleaned.replace(/我的+/g, '我的');

  // 2. 去除重复的"我的"和其他词的组合（如："我的我的我的" → "我的"）
  cleaned = cleaned.replace(/(我的)\s+/g, '我的 ');

  // 3. 去除多余的空格
  cleaned = cleaned.replace(/\s+/g, ' ').trim();

  // 4. 去除重复的标点符号（如："！！！？" → "！"）
  cleaned = cleaned.replace(/([！？。，、；：])\1+/g, '$1');

  // 5. 去除首尾的标点符号
  cleaned = cleaned.replace(/^[！？。，、；：，]+|[！？。，、；：，]+$/g, '');

  return cleaned;
};
```

**清理规则：**

| 原始输入 | 清理后 | 说明 |
|---------|--------|------|
| 我的我的我的成绩 | 我的成绩 | 去除重复的"我的" |
| 我的 我的 我的 成绩 | 我的成绩 | 去除多余的空格 |
| 怎么办！！！？ | 怎么办！ | 去除重复的标点 |
| ？挂科了会怎么样 | 挂科了会怎么样 | 去除首尾标点 |

## 💡 设计思路

### 为什么要清理输入？

1. **避免AI生成错误SQL**
   - 重复的词语可能让AI困惑
   - 导致生成的SQL不准确
   - 清理后的输入更清晰

2. **提升查询成功率**
   - 减少异常输入导致的错误
   - 提高SQL生成的准确率
   - 降低降级到普通聊天的概率

3. **改善用户体验**
   - 用户可能因为打字快或重复输入
   - 系统应该容错并自动修正
   - 不应该让用户为输入错误负责

### 为什么只清理部分内容？

1. **保留用户原始消息显示**
   - `userMessage.content` 使用原始`input`
   - 用户看到的是自己输入的内容
   - 不会感到困惑

2. **后端使用清理后的输入**
   - `cleanedInput` 用于意图识别和后端调用
   - 确保系统能正确处理
   - 不影响用户看到的内容

3. **只清理明显的错误**
   - 重复的词语："我的我的我的" → "我的"
   - 重复的标点："！！！" → "！"
   - 多余的空格："我的  我的" → "我的"
   - 不过度清理，保留用户的表达风格
