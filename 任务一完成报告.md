## 🚀 使用说明

### 1. 启动前端
```bash
cd frontend
npm install
npm run dev
```

### 2. 启动后端
```bash
cd backend
npm install
npm start
```

### 3. 访问页面
- **主页：** http://localhost:5173/
- **静态页：** http://localhost:5173/school-portal.html

### 4. 测试流程
1. 打开主页，看到学校介绍页面
2. 点击右下角圆形机器人图标
3. 浮窗展开，显示访客聊天界面
4. 输入问题测试基本问答
5. 尝试询问受限功能（如"我的成绩"）
6. 点击"登录"按钮跳转到登录页
7. 登录后自动切换到完整功能模式
