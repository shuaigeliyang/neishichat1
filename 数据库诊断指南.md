## 📋 诊断步骤

### 第1步：检查数据导入

1. **打开Bot编辑页面**
   ```
   https://www.coze.cn/space/7619963008504365090/bot/7619973585532174388
   ```

2. **进入"知识" → "表格"**

3. **点击"学生信息"表**

4. **查看记录数**
   - 应该显示：540条记录
   - 如果显示的记录数不对，说明数据没有完全导入

5. **浏览数据**
   - 手动滚动查看学院列
   - 确认是否有"电子工程学院"和"人工智能学院"

### 第2步：验证数据完整性

**在表格数据库页面执行以下检查**：

1. **搜索"电子工程学院"**
   - 使用搜索功能
   - 输入：电子工程学院
   - 看能否找到记录

2. **搜索"人工智能学院"**
   - 输入：人工智能学院
   - 看能否找到记录

3. **导出验证**
   - 如果表格支持导出
   - 导出为Excel
   - 在本地检查学院列的唯一值

### 第3步：重新导入数据（如果需要）

**如果发现数据确实不完整，重新导入**：

1. **准备Excel文件**
   - 确保本地导出的`学生信息.xlsx`包含所有数据
   - 检查文件大小：应该是43.57 KB左右
   - 检查行数：应该是541行（包含表头）

2. **删除旧表**
   - 在Coze的"知识" → "表格"
   - 找到"学生信息"表
   - 删除

3. **重新上传**
   - 点击"上传表格"
   - 选择`学生信息.xlsx`
   - 等待上传完成

4. **验证导入**
   - 检查记录数是否为540条
   - 搜索"电子工程学院"确认存在
   - 搜索"人工智能学院"确认存在

### 关键改进点

#### 1. **明确SQL查询规范**

```markdown
【数据库查询规范】
**重要**：查询列表、统计类问题时，必须使用以下SQL模式：

1. **查询唯一值列表**（获取所有不重复的值）：
   ```sql
   SELECT DISTINCT 列名 FROM 表名
   ```
   ❌ 不要使用：LIMIT（除非用户明确指定数量）
   ❌ 不要使用：TOP、SAMPLE等限制结果的函数

   ```sql
   SELECT COUNT(*) FROM 表名
   SELECT COUNT(DISTINCT 列名) FROM 表名
   ```

3. **完整查询**：
   ```sql
   SELECT * FROM 表名 WHERE 条件
   ```
```

- ❌ 不要截断结果
- ❌ 不要只显示部分数据
```

#### 3. **添加验证步骤**

```markdown
【查询验证】
如果查询结果与用户预期不符：
1. 说明实际查询到的结果

- 学生成绩表：包含学号、姓名、课程名称、课程类型、平时成绩、期中成绩、期末成绩、总评成绩、学分、学期、学年学期、授课教师

【数据库查询规范】
**重要**：查询列表、统计类问题时，必须遵循以下规范：

   ```sql
   SELECT DISTINCT 列名 FROM 表名

   ❌ 错误：SELECT DISTINCT 学院 FROM 学生信息 LIMIT 3（会截断结果！）
   ❌ 错误：SELECT DISTINCT 学院 FROM 学生信息 TOP 5（会截断结果！）

2. **统计数量**：
   ```sql
   SELECT COUNT(*) FROM 表名
   SELECT COUNT(DISTINCT 列名) FROM 表名
   ```
   ✅ 正确：SELECT COUNT(DISTINCT 学院) FROM 学生信息

3. **完整查询**：
   ```sql
   SELECT * FROM 表名 WHERE 条件
