## 🎯 重构目标

### 架构原则

1. **单一职责**：每个服务只做一件事
2. **清晰分层**：API层 → 服务层 → 数据层
3. **依赖注入**：配置集中管理
4. **错误隔离**：错误不影响整体
5. **可观测性**：日志、监控、追踪
6. **可扩展性**：易于水平扩展

### 技术栈

**后端核心**：
- Node.js + Express（保持）
- TypeScript（新增，类型安全）
- PM2（进程管理）

**服务架构**：
- API Gateway（统一入口）
- 业务服务（独立模块）
- Python Embedding服务（Docker化）

**数据层**：
- MySQL（关系数据）
- Redis（缓存和会话）
- 文件系统（文档存储）

**监控**：
- Winston（日志）
- Prometheus（指标）
- Grafana（可视化）

```javascript
class RAGService {
    constructor() {
        this.embeddingClient = new PythonEmbeddingClient();
        this.chatService = new AnthropicService();
        this.indexManager = new IndexManager();
    }

    async ask(question, options = {}) {
        // 1. 生成问题向量（本地）
        const questionEmbedding = await this.embeddingClient.getEmbedding(question);
        
        // 2. 检索相关文档
        const relevantDocs = await this.indexManager.search(questionEmbedding, options);
        
        // 3. 生成答案
        const answer = await this.chatService.chat(question, {
            context: relevantDocs,
            systemPrompt: this.getSystemPrompt()
        });
        
        return { answer, sources: relevantDocs };
    }
}
```

class RAGController {
    async ask(req, res, next) {
        try {
            const { question } = req.body;
            
            // 参数验证
            if (!question) {
                return res.status(400).json({
                    success: false,
                    error: '问题不能为空'
                });
            }
            
            // 调用服务
            const result = await ragService.ask(question);
            
            // 返回结果
            res.json({
                success: true,
                data: result
            });
        } catch (error) {
            next(error);
        }
    }
}
```

class AppError extends Error {
    constructor(message, statusCode = 500, code = null) {
        super(message);
        this.statusCode = statusCode;
        this.code = code;
    }
}

function errorHandler(err, req, res, next) {
    // 记录错误
    logger.error('Error:', err);
    
    // 开发环境返回详细错误
    if (process.env.NODE_ENV === 'development') {
        return res.status(err.statusCode || 500).json({
            success: false,
            error: err.message,
            stack: err.stack
        });
    }
    
    // 生产环境返回友好错误
    const statusCode = err.statusCode || 500;
    const message = err.message || '服务器内部错误';
    
    res.status(statusCode).json({
        success: false,
        error: message
    });
}
```
