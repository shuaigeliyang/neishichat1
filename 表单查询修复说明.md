## 🎯 修复的问题

**问题描述：** 用户问"有哪些表单可以下载？"时，被错误识别为数据库查询，导致返回403错误。

**正确行为：** 应该识别为表单查询，调用`/api/forms`接口，返回表单列表。

**新增handleFormQuery函数：**
```javascript
const handleFormQuery = async (question) => {
  try {
    const response = await axios.get('/api/forms', {
      headers: { Authorization: `Bearer ${localStorage.getItem('token')}` }
    });

    if (response.data.success && response.data.data) {
      const forms = response.data.data;

      // 按分类组织表单
      const formsByCategory = forms.reduce((acc, form) => {
        const category = form.category || '其他';
        if (!acc[category]) {
          acc[category] = [];
        }
        acc[category].push(form);
        return acc;
      }, {});

      // 构建表单列表消息
      let formContent = '📝 **系统可下载的表单列表**\n\n';
      // ... 格式化表单列表

      setMessages(prev => [...prev, {
        role: 'assistant',
        content: formContent,
        isFormList: true,
        forms: forms
      }]);
      return;
    }

    // 失败时回退到普通聊天
    await handleChatQuery(question);

  } catch (error) {
    await handleChatQuery(question);
  }
};
```

## 🔍 意图识别优先级

修复后的识别优先级：

1. **表单查询**（最高优先级）
   - 关键词：表单、表格、申请表、下载表单等

2. **数据库查询**（个人数据）
   - 关键词：我的、查询所有、查看、学生信息、教师信息等
   - 排除：包含"表单"的查询

3. **文档问答**（学生手册）
   - 关键词：规定、办法、重修、补考、转专业等

4. **普通聊天**（默认）
   - 其他所有查询
