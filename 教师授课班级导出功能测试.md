## 🔧 技术细节

### SQL查询逻辑

```sql
SELECT
  c.class_id AS '班级ID',
  c.class_name AS '班级名称',
  m.major_name AS '专业名称',
  m.major_code AS '专业代码',
  col.college_name AS '学院名称',
  col.college_code AS '学院代码',
  COUNT(DISTINCT co.course_id) AS '授课门数',
  GROUP_CONCAT(DISTINCT course.course_name SEPARATOR '、') AS '授课课程'
FROM course_offerings co
LEFT JOIN classes c ON co.class_id = c.class_id
LEFT JOIN majors m ON c.major_id = m.major_id
LEFT JOIN colleges col ON m.college_id = col.college_id
LEFT JOIN courses course ON co.course_id = course.course_id
WHERE co.teacher_id = ?  -- 当前登录教师的ID
  AND co.status = 1
GROUP BY co.class_id
ORDER BY c.class_name
```

### 权限验证

1. **身份验证：** 必须是已登录的教师用户
2. **权限验证：** userType === 'teacher'
3. **数据隔离：** 只查询当前教师（teacher_id = userId）的授课班级

## 🎯 前端调用示例

**Chat.jsx 中应该这样调用：**

```javascript
// 如果AI返回包含下载链接
if (response.downloadUrl) {
  // 显示下载按钮
  <button onClick={() => handleExport(response.downloadUrl)}>
    📥 下载授课班级信息
  </button>
}

// 下载处理函数
const handleExport = async (url) => {
  try {
    const response = await axios.get(url, {
      headers: {
        Authorization: `Bearer ${localStorage.getItem('token')}`
      },
      responseType: 'blob'
    });

    // 创建下载链接
    const blobUrl = window.URL.createObjectURL(new Blob([response.data]));
    const link = document.createElement('a');
    link.href = blobUrl;
    link.setAttribute('download', 'teaching_classes.xlsx');
    document.body.appendChild(link);
    link.click();
    link.remove();

    message.success('授课班级信息下载成功！');
  } catch (error) {
    message.error('下载失败：' + error.message);
  }
};
```

## 💡 常见问题

**Q1: 如果显示404错误？**
A: 后端服务可能没有重启，请重启后端服务。

**Q2: 如果显示403错误？**
A: 可能不是教师账号，请检查登录账号类型。

**Q3: Excel文件打开乱码？**
A: 使用Excel或WPS打开，CSV可能导致乱码，这是xlsx格式不会有问题。

**Q4: 下载的文件没有数据？**
A: 该教师没有授课班级，检查course_offerings表中是否有该教师的记录。
