## 🧪 第三阶段：后端API测试

### 测试目标
验证所有文档管理API接口

### 启动后端服务器

```bash
cd "E:\外包\教育系统智能体\backend"
npm run dev
```

服务器应该在 `http://localhost:3000` 启动

### API测试用例

#### 1. 获取所有文档

```bash
curl http://localhost:3000/api/documents
```

**预期响应：**
```json
{
  "success": true,
  "data": {
    "documents": [
      {
        "documentId": "policy_xxx",
        "name": "2025年本科学生手册-定",
        "status": "indexed",
        "statistics": {
          "totalChunks": 520
        }
      }
    ],
    "statistics": {
      "totalDocuments": 1,
      "indexedDocuments": 1
    }
  }
}
```

#### 2. 获取索引状态

```bash
curl http://localhost:3000/api/index/status
```

**预期响应：**
```json
{
  "success": true,
  "data": {
    "totalDocuments": 1,
    "totalChunks": 520,
    "documents": [...]
  }
}
```

#### 3. 上传新文档

```bash
# 准备一个测试文档（例如：奖学金评定办法.docx）
curl -X POST http://localhost:3000/api/documents/upload \
  -F "file=@相关文档/奖学金评定办法.docx" \
  -F "name=奖学金评定办法" \
  -F "description=学生奖学金评定相关办法" \
  -F "tags=奖学金,评定,奖励"
```

**预期响应：**
```json
{
  "success": true,
  "data": {
    "documentId": "policy_xxx",
    "name": "奖学金评定办法",
    "status": "pending"
  },
  "message": "文档上传成功，请开始处理"
}
```

#### 4. 处理文档

```bash
# 使用上一步返回的documentId
curl -X POST http://localhost:3000/api/documents/{documentId}/process
```

**预期响应：**
```json
{
  "success": true,
  "data": {
    "documentId": "policy_xxx",
    "status": "processing"
  },
  "message": "文档处理已启动"
}
```

#### 5. 查询文档状态

```bash
curl http://localhost:3000/api/documents/{documentId}/status
```

**预期响应：**
```json
{
  "success": true,
  "data": {
    "status": "indexed",
    "statistics": {
      "totalPages": 50,
      "totalChunks": 200,
      "indexedChunks": 200
    }
  }
}
```

#### 6. 删除文档

```bash
curl -X DELETE http://localhost:3000/api/documents/{documentId}
```

**预期响应：**
```json
{
  "success": true,
  "message": "文档已删除，相关索引已清理"
}
```

## 🐛 常见问题排查

### 问题1：迁移失败

**症状：** 迁移脚本报错

**排查：**
```bash
# 检查文件是否存在
ls -la "文档提取/2025年本科学生手册-定/"

# 检查文件内容
cat "文档提取/2025年本科学生手册-定/student_handbook_full.json" | head -20
```

### 问题2：API调用失败

**症状：** API返回错误

**排查：**
```bash
# 检查后端是否启动
curl http://localhost:3000/health

# 查看后端日志
# 应该有详细的错误信息
```

### 问题3：文档处理卡住

**症状：** 文档状态一直是"处理中"

**排查：**
```bash
# 检查文档目录
ls -la "文档提取/{文档名}/"

# 检查是否有student_handbook_full.json
cat "文档提取/{文档名}/student_handbook_full.json"
```

### 问题4：前台问答无结果

**症状：** 问答返回"未找到相关内容"

**排查：**
```bash
# 检查统一索引
cat "文档库/indexes/unified_index.json" | grep "totalChunks"

# 检查是否有文档已索引
curl http://localhost:3000/api/index/status
```
