### 3. 创建export_excel工具

#### 3.1 工具基本信息

- **工具名称**：`export_excel`
- **工具描述**：将数据导出为Excel文件。支持自定义表头、样式、列宽，可生成专业的Excel报表。适用于数据导出、报表生成、数据备份等场景。
- **运行时**：Python3

#### 3.2 完整代码（94行）

```python
import openpyxl
from openpyxl.styles import Font, Alignment, PatternFill
from openpyxl.utils import get_column_letter
from datetime import datetime
import os
from runtime import Args
from typings.export_excel.export_excel import Input, Output

def handler(args: Args[Input]) -> Output:
    """
    Excel导出工具 - 内师智能体系统的完美作品 (￣▽￣)ﾉ
    功能：将提供的数据生成为Excel文件
    """

    # 获取输入参数（添加对None的检查）
    input_data = args.input if args.input else {}
    data = input_data.get("data", [])
    filename = input_data.get("filename", f"export_{datetime.now().strftime('%Y%m%d_%H%M%S')}")
    sheet_name = input_data.get("sheet_name", "Sheet1")
    headers = input_data.get("headers", {})
    auto_column_width = input_data.get("auto_column_width", True)

    if not data:
        return {
            "success": False,
            "message": "❌ 错误：没有提供数据！请提供要导出的数据。"
        }

    # 创建工作簿
    wb = openpyxl.Workbook()
    ws = wb.active
    ws.title = sheet_name

    # 获取所有可能的列（从headers或第一条数据）
    if headers:
        columns = list(headers.keys())
    elif data and isinstance(data[0], dict):
        columns = list(data[0].keys())
    else:
        columns = []

    # 写入表头
    header_style = {
        'font': Font(name='微软雅黑', size=12, bold=True, color='FFFFFF'),
        'fill': PatternFill(start_color='4472C4', end_color='4472C4', fill_type='solid'),
        'alignment': Alignment(horizontal='center', vertical='center')
    }

    for col_idx, column in enumerate(columns, 1):
        cell = ws.cell(row=1, column=col_idx)
        header_text = headers.get(column, column) if headers else column
        cell.value = header_text
        cell.font = header_style['font']
        cell.fill = header_style['fill']
        cell.alignment = header_style['alignment']

    # 写入数据
    data_style = Alignment(horizontal='left', vertical='center')
    for row_idx, row_data in enumerate(data, 2):
        for col_idx, column in enumerate(columns, 1):
            cell = ws.cell(row=row_idx, column=col_idx)
            value = row_data.get(column, '') if isinstance(row_data, dict) else ''
            cell.value = value
            cell.alignment = data_style

    # 自动调整列宽
    if auto_column_width:
        for col_idx in range(1, len(columns) + 1):
            column_letter = get_column_letter(col_idx)
            max_length = 0
            for row in ws[1:len(data) + 2]:
                cell_value = str(row[col_idx - 1].value) if row[col_idx - 1].value else ''
                max_length = max(max_length, len(cell_value))
            adjusted_width = min(max_length + 2, 50)
            ws.column_dimensions[column_letter].width = adjusted_width

    # 保存文件
    temp_dir = "/tmp"
    filepath = os.path.join(temp_dir, f"{filename}.xlsx")
    wb.save(filepath)

    file_size = os.path.getsize(filepath) if os.path.exists(filepath) else 0

    return {
        "success": True,
        "message": f"✅ Excel文件生成成功！共导出 {len(data)} 条记录 (￣▽￣)ﾉ",
        "filename": f"{filename}.xlsx",
        "file_size": file_size,
        "record_count": len(data),
        "sheet_name": sheet_name,
        "filepath": filepath,
        "created_at": datetime.now().isoformat()
    }
```

#### 3.3 输入参数配置

**已配置参数**：
- **data** (Array<Object>)
  - 描述：要导出的数据数组，每个元素为对象，包含字段的键值对
  - 必填：是
  - 示例字段：id (String) - 示例字段，实际使用时可包含任意字段

**待配置参数**（可选）：
- **filename** (String) - 导出的文件名（不含扩展名）
- **sheet_name** (String) - Excel工作表名称
- **headers** (Object) - 表头字段映射（字段名→显示名称）
- **auto_column_width** (Boolean) - 是否自动调整列宽

#### 3.4 输出参数配置（8个字段）

1. **success** (Boolean) - Excel文件是否成功生成
2. **message** (String) - 操作结果消息或错误信息
3. **filename** (String) - 生成的Excel文件名（含.xlsx扩展名）
4. **file_size** (Number) - 生成的Excel文件大小（字节）
5. **record_count** (Integer) - 导出的数据记录总数
6. **sheet_name** (String) - Excel工作表名称
7. **filepath** (String) - Excel文件的完整保存路径
8. **created_at** (String) - Excel文件创建时间（ISO 8601格式）
