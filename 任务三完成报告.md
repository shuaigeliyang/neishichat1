## 🎯 清理规则详解

### 1. 孤立标点符号移除
**规则：** 单独一行的标点符号会被移除
```
。
，
、
```
→ 全部移除

### 2. 标点符号标准化
**规则：** 英文标点转换为中文标点
```
, → ，
; → ；
: → ：
. → 。
? → ？
! → ！
( → （
) → ）
```

### 3. 重复标点符号清理
**规则：** 多个重复标点符号合并为一个
```
，，， → ，
。。。 → 。
；；； → ；
```

### 4. 空格清理
**规则：** 移除标点符号前后的空格
```
学生 在学校 → 学生在学校
修完 专业 人才培养 → 修完专业人才培养
```

### 5. 断词修复
**规则：** 修复被换行符打断的词语
```
办公\n室 → 办公室
教学科研\n工作 → 教学科研工作
升、跳、留、降\n级 → 升、跳、留、降级
```

### 6. PDF提取问题修复
**规则：** 针对PDF解析产生的特定问题进行修复
```
办公
室 → 办公室
（
） → （）
"
" → "
```

## 🔧 技术实现

### 清理流程
```
原始文本
  ↓
1. 移除孤立标点符号
  ↓
2. 标准化换行符
  ↓
3. 标准化标点符号
  ↓
4. 移除多余空格
  ↓
5. 清理重复标点
  ↓
6. 修复断词
  ↓
7. 移除多余空行
  ↓
8. 段落重组
  ↓
清理后的文本
```

### 正则表达式规则
```javascript
// 孤立标点符号
/^[。，、；：,.;:!?！？，,、\s]+$/

// 重复标点符号
/\（[、。，]+\)/g
/[，。；：！？、]+[，。；：！？、]+/g

// PDF断词修复
/办公\n室/g
/教学科研\n工作/g
/升、跳、留、降\n级/g
```
