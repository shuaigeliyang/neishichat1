============================================
  启动脚本使用说明
============================================

已修复的问题：
  ✅ Python路径: D:\ano\python.exe
  ✅ 使用绝对路径，避免路径错误
  ✅ 所有脚本使用Windows正确的 >/dev/null 语法
  ✅ 添加 pause 命令防止窗口闪退

============================================
使用方法：
============================================

启动所有服务：
  双击：启动所有服务.bat
  
  会依次启动：
  1. Python Embedding Service (5001) - 等待15秒
  2. Main Backend (3000) - 等待10秒
  3. Admin Backend (3005) - 等待10秒
  4. Main Frontend (5173) - 等待8秒
  5. Admin Frontend (5176) - 等待8秒
  
  完成后自动打开浏览器：
  - http://localhost:5173 (主系统)
  - http://localhost:5176 (后台管理)

停止所有服务：
  双击：停止所有服务.bat
  
  会强制结束所有 Node.js 和 Python 进程

重启所有服务：
  双击：重启所有服务.bat
  
  先停止所有服务，等待5秒，然后自动启动

============================================
注意事项：
============================================

1. 每个服务会在独立的命令行窗口中运行
2. 可以看到每个服务的实时日志
3. 不要关闭服务窗口，否则服务会停止
4. 如果端口被占用，先运行停止脚本
5. 确保已安装 Node.js 和 Python

============================================
服务端口说明：
============================================

  Python Embedding: 5001 (最重要，其他依赖它)
  Main Backend:     3000
  Admin Backend:    3005
  Main Frontend:    5173
  Admin Frontend:   5176
