## 优势对比

| 特性 | 旧方案（智谱API） | 新方案（本地+Minimax） |
|------|------------------|------------------------|
| Embedding成本 | 按调用计费 | 免费 |
| Embedding限流 | 经常429错误 | 无限流 |
| Embedding速度 | ~500ms/请求 | ~50ms/请求（快10倍） |
| Chat API | glm-4-flash | Claude Sonnet |
| Chat质量 | 一般 | 显著提升 |
| 配置复杂度 | 需要维护多个API Key | 单一Minimax Key |
